## Blog site
- Этот проект Python представляет собой типовой блог-проект, в котором пользователь имеет возможность просматривать блоги, регистрироваться, авторизироваться, добавлять собственные тексты, фотографии. 
- Админка представляет собой расширенную модель Django.
### Используемые технологии:
- язык Python,
- фреймворк Django,
- SQL-запросы,
- СУБД – SQLite3,
- ListView, DetailView,
- CreateView, Mixins.


###### Как запускать:
- в терминале Pycharm: python manage.py runserver
- открыть проект в браузере: http://127.0.0.1:8000/

###### Вход в админку:
- http://127.0.0.1:8000/admin/
- superuser name: admin
